CRM Project 

used virtualenv to create virtual environment
Steps :
-> pip install virtual env (because vtualenv doesnot comes as default)
-> move to the directory you want to create the virtual environment 
-> virtualenv [dir]
-> activate the environment
-> pip install django


Creating Project and app in Django

-> django-admin startproject CRM
-> move a level into the project directory
-> manage.py startapp account


creating template
